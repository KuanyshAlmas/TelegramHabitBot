# Habit Tracker Telegram Bot

Telegram-бот для отслеживания привычек с системой страйков, марафонами и аналитикой.

## Возможности

- **Два типа привычек**: Галочка (да/нет) и Цифровая (с целью)
- **Категории**: Организация привычек по папкам
- **Страйки (огоньки)**: Награда за последовательность
- **Правило 10 минут**: Автоматические уведомления с таймером
- **Итоги дня в 23:59**: Автоматический расчет страйков
- **Марафоны**: Соревнования с друзьями
- **Аналитика**: Графики и статистика за любой период
- **Еженедельные/ежемесячные отчеты**: Автоматически

## Установка

### 1. Клонирование и зависимости

```bash
cd habit_tracker
pip install -r requirements.txt
```

### 2. Получение токена бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

### 3. Настройка

```bash
cp .env.example .env
# Отредактируйте .env и вставьте ваш токен
```

### 4. Запуск

```bash
python bot.py
```

## Команды бота

- `/start` - Начать работу с ботом

### Меню

- **Мои привычки** - Список привычек
- **Внести результат** - Быстрый ввод
- **Моя статистика** - Аналитика
- **Марафоны** - Соревнования
- **Категории** - Управление папками
- **Настройки** - Время уведомлений

## Как это работает

### Уведомления
1. Вы выбираете время уведомлений (например, 08:00, 14:00, 21:00)
2. В это время бот присылает напоминание
3. У вас есть 10 минут на ответ
4. Если не ответили - засчитывается 0

### Страйки
- Выполнил цель = страйк +1
- Не выполнил = страйк сгорает
- Рассчитывается в 23:59

### Марафоны
1. Создайте марафон с датами
2. Добавьте привычки
3. Поделитесь кодом с друзьями
4. Соревнуйтесь в таблице лидеров

## Часовой пояс

Бот работает по времени Казахстана (Asia/Almaty, UTC+5).
